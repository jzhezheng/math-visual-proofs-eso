# Visual Proofs for Secondary Mathematics / Demostracions Visuals per a les Matemtiques de Secundria

 **Languages:** [English](#english) | [Catal](#catal)

Mathematical visual proofs for secondary education (ESO) created with **ManimCE**, developed as part of a Treball de Recerca (high school research project) in Catalonia.

## English

This repository contains animations created with ManimCE for a high school research project (Treball de Recerca) titled **"Visual Mathematical Proofs for Secondary Education: An Accessible Approach with Digital Tools"**.

This project demonstrates, through **animations created with ManimCE**, how to construct essential mathematical formulas and theorems from **lower secondary education (ESO level)** step by step.

> ** Educational Level Note**. These animations focus on mathematical concepts taught in **ESO** (Educaci贸 Secundria Obligat貌ria), which corresponds to lower secondary education (typically ages 12-16) in the Spanish educational system. The content covers key concepts from algebra, geometry, and fundamental proofs appropriate for this level.

###  Repository Structure

```
 docs/                         # Documentation (EN & CA)
 src/                          # Python source code (ManimCE scenes)
 requirements.txt              # Python dependencies
```

###  Installation & Usage

1. **Install Python 3.9+**

2. **Clone the repository**

   ```bash
   git clone https://github.com/jzhezheng/math-visual-proofs-eso
   ```

3. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

4. **Run an animation**
   ```bash
   manim -p -qh src/algebraic_identities.py
   ```

 _Detailed instructions: [Installation Guide](docs/en/installation_guide.md)_
